# TO DO App

A simple **TO DO application** built with **Django** to manage tasks. Users can add tasks, view all tasks, mark tasks as done, and delete tasks.  



##  Features

- Add new tasks with a description and deadline.  
- View all tasks in a table format.  
- Mark tasks as completed.  
- Delete tasks when no longer needed.  
- Responsive UI using **Bootstrap**.




##  Pages

1. **Home Page** (`index.html`)  
   - Quick links to add a task or view all tasks.

2. **Add Task** (`addtask.html`)  
   - Form to add a task with description and deadline.  
   - Shows success form after submission.

3. **View Tasks** (`viewtask.html`)  
   - Displays all tasks in a table.  
   - Includes buttons to mark tasks as done or delete them.



##  Technologies Used

The project was built using the following technologies:  

- **Python 3** – The programming language used to build the backend logic.  
- **Django** – The web framework that handles routing, views, and models.  
- **Bootstrap 5** – For responsive and modern UI styling.  
- **HTML / CSS** – For structuring and styling the pages.  
- **SQLite** – The default database used for storing tasks.
